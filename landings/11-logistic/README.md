# ГрузПоРоссии — лендинг грузоперевозок

Одностраничный адаптивный лендинг для компании по грузоперевозкам по России.
Чистый HTML5 + CSS3 + JavaScript, без фреймворков и сборки. Готов к размещению
на любом хостинге — просто загрузите файлы.

## Структура

```
11-logistic/
├── index.html        — разметка страницы (все секции, SEO, Open Graph, микроразметка)
├── style.css         — стили: дизайн-система, адаптив, анимации
├── script.js         — калькулятор, маска телефона, валидация формы, меню, анимации
├── favicon/
│   └── favicon.svg   — иконка сайта (грузовик)
├── images/           — локальные изображения (по умолчанию фото подгружаются с CDN Unsplash)
├── icons/            — иконки (используются встроенные inline-SVG прямо в разметке)
└── README.md
```

## Возможности

- **Главный экран** с фоновой фотографией, заголовком, ценой 7 ₽/км и кнопками «Позвонить» / «Заказать перевозку».
- **Калькулятор стоимости** — пользователь вводит километраж, цена считается мгновенно (7 ₽ × км) с плавной анимацией.
- **Инфографика кузова** (SVG): длина 4,20 м, высота 2,5 м.
- **Карточки**: преимущества, что перевозим (с фото), водители, отзывы.
- **Форма заявки** с валидацией и маской телефона (демо-режим — показывает успех).
- **Мессенджеры**: кнопки WhatsApp и Telegram, плавающая кнопка WhatsApp, кнопка «Наверх», фиксированная панель звонка на мобильных.
- **Адаптивность**: корректно отображается на смартфонах (iPhone/Android), планшетах, ноутбуках и больших мониторах.
- **Производительность**: один CSS-файл, один JS-файл, шрифт Onest и фото — через CDN с `preconnect`, ленивая загрузка изображений (`loading="lazy"`), `fetchpriority` для hero.
- **Доступность**: семантическая разметка, `aria`-атрибуты, поддержка клавиатуры, `prefers-reduced-motion`, видимый фокус.
- **SEO**: `title`, `description`, `keywords`, Open Graph, Twitter Card, canonical, JSON-LD (`MovingCompany`).

## Фотографии

Тематические фотографии подгружаются с CDN **Unsplash** по прямым ссылкам в `index.html`.
Они отображаются сразу после загрузки сайта на хостинг, без битых ссылок. У каждого
изображения есть `onerror`-обработчик: если фото вдруг недоступно, блок аккуратно
показывает фирменный тёмно-синий фон вместо «сломанной» картинки.

Чтобы хостить фото локально (без зависимости от внешнего CDN) — скачайте их в папку
`images/` и замените ссылки `https://images.unsplash.com/...` на относительные пути
(например, `images/hero.jpg`).

## Подключение реальной отправки заявок

Сейчас форма работает в демо-режиме (показывает сообщение об успехе, никуда не
отправляет данные). Чтобы получать заявки, подключите в `script.js` (обработчик
`form submit`) один из вариантов:

- **Formspree** / **Getform** — `fetch('https://formspree.io/f/XXX', {...})`;
- **Telegram-бот** — отправка через Bot API на ваш `chat_id`;
- серверный обработчик на вашем хостинге.
