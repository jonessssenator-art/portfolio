# Как включить приём заявок прямо в Telegram (15 минут, бесплатно)

Сейчас заявка с сайта открывает чат менеджера, и покупатель вставляет текст сам.
После настройки по этой инструкции заявки будут **сами падать сообщением в Telegram** — покупателю ничего вставлять не надо.

## Шаг 1. Создать бота (2 минуты)

1. Открой в Telegram бота **@BotFather**
2. Отправь команду `/newbot`
3. Придумай имя, например: `Time Brand Заявки`
4. Придумай логин бота, он должен кончаться на `bot`, например: `timebrand_leads_bot`
5. BotFather пришлёт **токен** — длинную строку вида `1234567890:AAE...xyz`. Сохрани её, никому не показывай

## Шаг 2. Узнать свой chat_id (1 минута)

1. Открой в Telegram бота **@userinfobot** и нажми Start
2. Он пришлёт твой **Id** — число вида `123456789`. Сохрани
3. Открой СВОЕГО нового бота (из шага 1) и нажми Start — иначе он не сможет тебе писать

## Шаг 3. Завести бесплатный Cloudflare (5 минут)

Это «прослойка», которая прячет токен бота от посторонних (в код сайта токен вставлять нельзя — его украдут).

1. Зайди на **workers.cloudflare.com** → Sign Up (нужна только почта)
2. В меню слева: **Workers & Pages → Create → Create Worker**
3. Назови воркер `timebrand-leads` → **Deploy**
4. Нажми **Edit code**, сотри всё и вставь содержимое файла `worker.js` (лежит рядом с этой инструкцией) → **Deploy**

## Шаг 4. Вставить секреты (2 минуты)

1. Вернись на страницу воркера → вкладка **Settings → Variables and Secrets**
2. Добавь две переменные (тип — **Secret**):
   - имя `BOT_TOKEN`, значение — токен из шага 1
   - имя `CHAT_ID`, значение — число из шага 2
3. Сохрани

## Шаг 5. Подключить к сайту (1 минута)

1. Скопируй адрес воркера — он вида `https://timebrand-leads.ИМЯ.workers.dev`
2. Открой `index.html` сайта, найди строку:
   ```js
   var LEAD_ENDPOINT='';
   ```
3. Вставь адрес между кавычек:
   ```js
   var LEAD_ENDPOINT='https://timebrand-leads.ИМЯ.workers.dev';
   ```
4. Загрузи сайт заново (закоммить и запушить)

## Проверка

Открой сайт, заполни форму, отправь. В Telegram должно прийти сообщение вида:

> 🔔 Заявка с сайта TIME BRAND
> Связь: +7 900 …
> Модель: SEIKO 5 SPORTS GMT — 29 000 ₽

Если что-то не пришло — проверь, что нажал Start у своего бота (шаг 2.3) и что секреты вписаны без пробелов.

**Важно:** даже если воркер сломается, сайт не потеряет заявку — форма автоматически откатится на старый способ (открыть чат менеджера с готовым текстом).
