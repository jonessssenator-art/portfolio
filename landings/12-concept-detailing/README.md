# Concept Detailing — лендинг детейлинг-центра (Астрахань)

Премиальный одностраничный сайт (Landing Page) детейлинг-центра **Concept Detailing**.
Чистый стек: **HTML5 + CSS3 + JavaScript**, без фреймворков и сборки. Полностью адаптивный,
готов к загрузке на любой хостинг «как есть».

## Дизайн
- Стиль **Luxury auto**: матовый чёрный фон, графитовые поверхности, золотые акценты, glassmorphism.
- Шрифты: **Oswald** (заголовки) + **Manrope** (текст) — Google Fonts.
- Плавные анимации появления, hover-эффекты, блики на кнопках, дыхание золотого свечения.
- Уважение к `prefers-reduced-motion` и работоспособность без JavaScript (graceful degradation).

## Разделы
1. Главный экран (Hero) с фоновым авто и CTA «Записаться» / «Рассчитать стоимость»
2. Лента доверия
3. О центре
4. Каталог услуг с ценами — **9 категорий в табах**: детейлинг-мойка, автомойка (матрица по типу кузова),
   тонировка, антигравийная защита (PPF), оклейка винилом, полировка, керамика, химчистка, доп. услуги
5. Калькулятор стоимости (услуга × класс авто)
6. Почему выбирают нас
7. Галерея **До / После** (интерактивные слайдеры) + лента работ
8. Наша команда (мастера + менеджеры с телефонами)
9. Отзывы
10. FAQ (аккордеон)
11. Контакты + форма онлайн-записи + каналы (Позвонить / WhatsApp / Instagram)
12. Футер
13. Плавающие кнопки: WhatsApp, «Наверх», мобильная панель «Позвонить / Записаться»

## Функциональность (JavaScript)
- Табы каталога услуг.
- Калькулятор: ориентировочная стоимость = базовая цена услуги × коэффициент класса авто, с анимацией счётчика.
- Слайдеры «До / После» (мышь, тач, клавиатура).
- FAQ-аккордеон.
- Мобильное меню с фокус-ловушкой и блокировкой скролла.
- Маска телефона, валидация и отправка формы.
- Скролл-спай навигации, кнопка «Наверх», анимации появления с самовосстановлением.

## Важно для заказчика (что доработать перед боевым запуском)
- **Форма онлайн-записи отправляет заявку в WhatsApp менеджеру (Мухтар, 8 999 646-67-79).**
  При нажатии «Записаться» собирается текст заявки (имя, телефон, услуга, комментарий) и открывается чат
  WhatsApp с готовым сообщением — клиенту остаётся нажать «отправить». Работает на статичном хостинге без сервера.
  Чтобы сменить номер — поменяйте `WA_NUMBER` в `script.js`. При желании можно переключить на Telegram-бот,
  Formspree (email) или CRM.
- **Цены — ориентировочные** по рынку Астрахани (2025–2026). Перед публикацией сверьте с вашим прайсом.
- **Фото галереи «До / После» — демонстрационные** (один кадр с эффектом затемнения «до»). Замените на
  реальные парные фото ваших работ — структура слайдера уже готова.
- Фотографии автомобилей — со стоковой библиотеки Unsplash (свободная лицензия), скачаны локально в `images/`.
- При желании добавьте: адрес и карту, реквизиты, страницу политики конфиденциальности, реальные отзывы.

## Запуск
Откройте `index.html` в браузере или загрузите папку целиком на любой хостинг (GitHub Pages, обычный
shared-хостинг, Netlify и т.п.). Интернет нужен только для подгрузки Google Fonts.
