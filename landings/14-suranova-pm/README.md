# Мария Суранова — перманентный макияж (лендинг #14)

Премиальный одностраничный сайт мастера перманентного макияжа в Вологде:
услуги и цены, работы, до/после, отзывы, обучение, FAQ и **онлайн-запись
с отправкой заявки в WhatsApp**.

Чистый HTML/CSS/JS без сборки и зависимостей. Открывается двойным кликом,
разворачивается на GitHub Pages / Vercel / любой хостинг.

## Структура

```
14-suranova-pm/
├── index.html      ← сам сайт (для смены контента править не нужно)
├── content.js      ← ВСЕ ДАННЫЕ: цены, услуги, отзывы, FAQ, контакты (правьте здесь)
├── admin.html      ← панель редактирования в браузере (правки без кода)
├── images/         ← картинки-заглушки (.svg), замените на реальные фото
└── README.md
```

## Как менять контент

Два способа — выберите удобный:

**1. Через файл `content.js`** — откройте в «Блокноте», меняйте текст в кавычках,
сохраните, обновите страницу. Там цены, услуги, отзывы, вопросы-ответы, контакты,
блок обучения и показатели в первом экране.

**2. Через админку `admin.html`** — откройте её в браузере, меняйте поля,
жмите **«Сохранить»** (правки сразу видны на сайте в этом браузере — через localStorage).
Чтобы правки увидели все посетители — нажмите **«Скачать content.js»** и замените
этим файлом старый `content.js` в папке сайта.

## Как заменить фото

Положите свои файлы в папку `images/` и укажите их имена:

- Фото мастера в первом экране — `images/hero-master.svg` (в `index.html`)
- Фото в блоке «О мастере» — `images/about-master.svg`
- До / после — `images/before.svg`, `images/after.svg`
- Обучение — `images/training.svg`
- Галерея — `images/gallery-1.svg … gallery-8.svg` (список в `content.js` → `gallery`)

Можно заменить `.svg` на свои `.jpg/.webp` — поправьте имя файла в `index.html`
или в `content.js`. Рекомендуемые пропорции: портреты 4:5, до/после 4:3, галерея — любые.

> Текущие изображения — фирменные заглушки в стиле бренда. Замените их на реальные
> фото работ и мастера перед публикацией.

## Онлайн-запись → WhatsApp

Клиент выбирает дату в календаре, время, услугу, вводит имя, телефон и комментарий.
По кнопке формируется сообщение и открывается WhatsApp мастера:

```
Здравствуйте! Хочу записаться.
Имя: …
Услуга: …
Дата: …
Время: …
Телефон: …
Комментарий: …
```

Рабочие дни и интервалы времени настраиваются в `content.js` (`bookingDays`, `slots`)
или в админке. Кнопки «Записаться» у каждой услуги подставляют услугу в форму.

## Блок разработчика

Ссылка на портфолио вынесена в `content.js` → `config.portfolioLink`
(по умолчанию — портфолио студии). Меняется в одном месте.

## Публикация

- **GitHub Pages:** папка лежит в репозитории портфолио, адрес вида
  `…/portfolio/landings/14-suranova-pm/`.
- **Vercel / Netlify:** перетащите папку — деплой статики без настроек.
- Локально: просто откройте `index.html` в браузере.

## Технологии и качество

- Один файл, минимум JS, ленивая загрузка изображений — быстрая загрузка.
- SEO: title/description/keywords, Open Graph, Twitter Card, Schema.org (BeautySalon), canonical.
- Доступность: семантика, `aria`-атрибуты, фокус, поддержка `prefers-reduced-motion`.
- Адаптив: мобильные, планшеты, десктоп.

> ⚠️ Услуга имеет противопоказания. Формулировки на странице — маркетинговые,
> не медицинские; финальные тексты согласуйте под практику мастера.
