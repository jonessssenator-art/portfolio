# Мария Суранова — перманентный макияж (лендинг #14)

Премиальный одностраничный сайт мастера перманентного макияжа в Вологде:
услуги и цены, работы, до/после, отзывы, обучение, FAQ и **онлайн-запись
с отправкой заявки в WhatsApp** (компактный календарь + выбор времени).

Один самодостаточный файл `index.html` — без сборки, зависимостей и внешних
данных. Открывается двойным кликом, разворачивается на GitHub Pages / Vercel /
любой хостинг.

## Структура

```
14-suranova-pm/
├── index.html   ← весь сайт + данные (правки здесь)
├── images/      ← фото (.jpg) — замените на реальные снимки мастера
└── README.md
```

## Как менять контент (без программиста)

Все данные — цены, услуги, отзывы, вопросы-ответы, контакты, тексты обучения —
лежат в одном месте: откройте `index.html`, найдите блок

```js
window.SURANOVA_CONTENT = { ... }
```

и меняйте значения в кавычках. Не удаляйте кавычки, запятые и скобки.

## Как заменить фото

Положите свои файлы в `images/`, сохранив имена (или поправьте имя в коде):

- Первый экран — `images/hero-master.jpg`
- О мастере — `images/about-master.jpg`
- До / после — `images/before.jpg`, `images/after.jpg`
- Обучение — `images/training.jpg`
- Галерея — `images/gallery-1.jpg … gallery-8.jpg` (подписи в блоке `gallery`)
- Превью для соцсетей — `images/og-cover.jpg`

Пропорции: портреты 4:5, до/после 4:3, галерея — любые.

> Сейчас стоят стоковые фото (Unsplash, свободная лицензия) как временные —
> замените их на реальные работы и фото мастера перед публикацией.

## Онлайн-запись → WhatsApp

Клиент выбирает дату в календаре, время, услугу, вводит имя, телефон и
комментарий → формируется сообщение и открывается WhatsApp мастера:

```
Здравствуйте! Хочу записаться.
Имя: … · Услуга: … · Дата: … · Время: … · Телефон: … · Комментарий: …
```

Рабочие дни и интервалы — в `config` (`bookingDays`, `slots`). Кнопки
«Записаться» у услуг подставляют выбранную услугу в форму.

## Технологии

- Одна страница, минимум JS, ленивая загрузка изображений — быстрая загрузка.
- Шрифты: **Onest** (заголовки) + **Manrope** (текст) + подпись Marck Script.
- SEO: title/description/keywords, Open Graph, Twitter Card, Schema.org (BeautySalon), canonical.
- Доступность: семантика, `aria`, фокус, `prefers-reduced-motion`. Адаптив: мобильные/планшет/десктоп.
- Блок разработчика: ссылка в `config.portfolioLink`.

> ⚠️ Услуга имеет противопоказания. Тексты — маркетинговые, не медицинские;
> финальные формулировки согласуйте под практику мастера.
