# Каталог 3D-эффектов

Библиотека готовых 3D/визуальных эффектов для лендингов.

## Структура

- `effects/` — каждый эффект = один самодостаточный HTML-файл (`NN-слаг.html`), открывается в браузере без сборки.
- `index.html` — витрина: список всех эффектов с живым предпросмотром (iframe). При добавлении эффекта дописать его в массив `EFFECTS` внутри `index.html`.

## Правила добавления эффекта

1. Сохранить код как `effects/NN-короткое-имя.html` (NN — порядковый номер: 01, 02, …).
2. Файл должен быть самодостаточным: весь CSS/JS инлайном; внешние CDN (three.js и т.п.) допустимы.
3. В начало файла — комментарий-паспорт:
   ```html
   <!--
   Эффект: <название>
   Технология: CSS 3D / three.js / WebGL / canvas …
   Зависимости: нет | CDN <url>
   Куда подходит: hero, карточки, фон, кнопки …
   Добавлен: <дата>
   -->
   ```
4. Добавить запись в `EFFECTS` в `index.html`.

## Использование

По запросу эффект берётся из `effects/` и встраивается в нужный лендинг (адаптируя цвета/размеры под проект).
